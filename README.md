# README file

Πρωθυπουργός για μια μέρα: Σύστημα Διαχείρισης Κρατικού Προϋπολογισμού

Αυτό το έργο υλοποιεί ένα σύστημα επισκόπησης και επεξεργασίας του κρατικού προϋπολογισμού, επιτρέποντας στον χρήστη (τον "Πρωθυπουργό για μια μέρα") να αναλύει τα στοιχεία, να εισάγει αλλαγές και να ελέγχει τους περιορισμούς, όλα μέσω της γραμμής εντολών.
