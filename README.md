Σύστημα Διαχείρισης Κρατικού Προϋπολογισμού / State Budget Management System 

Επισκόπηση Εφαρμογής / Project Overview
Η παρούσα εφαρμογή είναι ένα ολοκληρωμένο πληροφοριακό σύστημα σε περιβάλλον Java Swing για την παρακολούθηση, ανάλυση και τροποποίηση των δεδομένων του ελληνικού κρατικού προϋπολογισμού. This application is a comprehensive information system built with Java Swing for monitoring, analyzing, and modifying the Greek State Budget data.
