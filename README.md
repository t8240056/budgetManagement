# README file

Πρωθυπουργός για μια μέρα: Σύστημα Διαχείρισης Κρατικού Προϋπολογισμού

Αυτό το έργο υλοποιεί ένα σύστημα επισκόπησης και επεξεργασίας του κρατικού προϋπολογισμού, επιτρέποντας στον χρήστη (τον "Πρωθυπουργό για μια μέρα") να αναλύει τα στοιχεία, να εισάγει αλλαγές και να ελέγχει τους περιορισμούς, όλα μέσω της γραμμής εντολών.

Λειτουργικότητα (Ελάχιστες Απαιτήσεις)

Η εφαρμογή υλοποιεί πλήρως τις ακόλουθες ελάχιστες απαιτήσεις:

### 1. Εμφάνιση Στοιχείων Προϋπολογισμού
* Ο χρήστης μπορεί να επιλέξει να εμφανίσει τον πλήρη προϋπολογισμό, διαχωρισμένο σε **Έσοδα** και **Έξοδα**.
* Η δομή του προϋπολογισμού είναι ιεραρχική, επιτρέποντας την επισκόπηση κατηγοριών και υποκατηγοριών.

### 2. Εισαγωγή και Διαχείριση Αλλαγών
* Η εφαρμογή παρέχει διαδραστικό μενού για την **εισαγωγή αλλαγών** σε συγκεκριμένες γραμμές του προϋπολογισμού.
* Όλες οι αλλαγές καταγράφονται και αποθηκεύονται στη μνήμη, διατηρώντας το αρχικό ποσό για σύγκριση.

### 3. Έλεγχοι Περιορισμών
* Πριν από την εφαρμογή οποιασδήποτε αλλαγής, εκτελούνται **αυστηροί έλεγχοι περιορισμών** (π.χ. να υπάρχουν αριθμητικοί περιορισμοί σε κάποια από τα υπουργεία).
* Εάν ένας περιορισμός παραβιαστεί, η αλλαγή απορρίπτεται και εμφανίζεται σχετικό μήνυμα λάθους.

### 4. Επισκόπηση Εισαχθέντων Αλλαγών
* Υπάρχει ειδική επιλογή στο μενού που εμφανίζει **μόνο** τις γραμμές του προϋπολογισμού που έχουν τροποποιηθεί από τον χρήστη, μαζί με το αρχικό τους ποσό και το νέο, ώστε να είναι άμεσα ορατές οι παρεμβάσεις.
