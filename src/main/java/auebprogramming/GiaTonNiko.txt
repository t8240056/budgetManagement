Λοιπον μπέμπη Νίκο άκου

---ExpenseDisplay---

* ExpenseDisplay(String, String) -> Constructor -> Φορτώνει τα δεδομένα από τα δύο
    CSV (categoriesFile, ministriesFile) κατά τη δημιουργία του αντικειμένου, δηλαδή 
    εδώ σαν ορίσματα θα βάλεις τα 2 csv. Αν έχουμε και τα csv άλλων ετών θα
    μπορείς να τα βάζεις και αυτά σαν ορίσματα για να εμφανίζει και από άλλες χρονιές.

* getCategoriesReport(String) -> Δημιουργεί Αναφορά ανά Δαπάνη -> Επιστρέφει το πλήρες,
    μορφοποιημένο String του πίνακα "Πιστώσεις κατά Μείζονα Κατηγορία Δαπάνης" για τον
    συγκεκριμένο τύπο προϋπολογισμού (π.χ., "ΚΡΑΤΙΚΟΣ")

* getMinistriesReport(String) -> Δημιουργεί Αναφορά ανά Φορέα -> Επιστρέφει το πλήρες,
    μορφοποιημένο String του πίνακα "Πιστώσεις Συνολικά κατά Φορέα" για τον συγκεκριμένο
    τύπο προϋπολογισμού

* getAllExpenseReports() -> ΝΕΑ ΜΕΘΟΔΟΣ: Ολα Μαζί -> Εκτελεί τις 6 κλήσεις 
    (getCategoriesReport και getMinistriesReport για τους 3 τύπους Πρ., είναι αυτό που
    σου είπα που είναι περίπλοκο και τώρα αντί να καλείς 6 μεθόδους, θα καλείς αυτή)
    και επιστρέφει έναν πίνακα String[] με τις 6 αναφορές. ΕΣΥ ΑΥΤΟ ΘΑ ΚΑΛΕΙΣ!

* updateCategoryAmount(...) -> Εισαγωγή Αλλαγών ->Ενημερώνει το ποσό ενός συγκεκριμένου
    κωδικού δαπάνης στον εσωτερικό πίνακα categoriesData. ΔΕΝ ΣΕ ΝΟΙΑΖΕΙ ΑΚΟΜΑ


---ExpenseManager---

* ExpenseManager(String) -> Constructor ->	Φορτώνει τα δεδομένα των Κατηγοριών Δαπανών 
    από το CSV κατά τη δημιουργία του αντικειμένου. Πάλι και εδώ αν του βάλεις csv από
    άλλες χρονιές (πάντα με την ίδια μορφοποίηση) θα κάνει την ίδια δουλειά και για 
    αυτά τα έτη.

* getCategoryListReport() -> Λίστα Επιλογών -> Επιστρέφει ένα συνοπτικό String 
    (Κωδικός / Όνομα) όλων των δαπανών. Ιδανικό για να γεμίσει λίστες επιλογής (ComboBoxes)

* getFullExpensesReport() -> Πλήρης Λίστα Δαπανών -> Επιστρέφει ένα μορφοποιημένο String 
    με τον Κωδικό, την Περιγραφή και το ποσό του Κρατικού Προϋπολογισμού (μία από τις 6 
    αναφορές), μαζί με το σύνολο

* getExpenseDetailsReport(String...) -> Λεπτομέρειες Κωδικών -> Επιστρέφει ένα 
    μορφοποιημένο String με τις πλήρεις λεπτομέρειες (Κρατικός, Τακτικός, Επενδύσεων) για 
    τους κωδικούς που ζήτησε ο χρήστης


ΤΙ ΚΑΝΕΙΣ ΕΣΥ?
Δημιουργείς τα αρχικά αντικείμενα (μία φορά):

final String categoriesFile = "expense_categories_2025.csv";
final String ministriesFile = "expense_ministries_2025.csv";
//Ξαναλέω εδώ βάζεις από όποια χρονιά θες

// Δημιουργία αντικειμένων
final ExpenseDisplay display = new ExpenseDisplay(categoriesFile, ministriesFile);
final ExpenseManager manager = new ExpenseManager(categoriesFile);



Για να εμφανίσεις τα έξοδα όλα μαζί καλείς τη μέθοδο getAllExpenseReports():

// 1. Καλούμε τη μέθοδο
String[] allReports = display.getAllExpenseReports();

// 2. Χρησιμοποιούμε τα Strings (για JTextArea, κτλ.)
// Αναφορές Δαπάνης
String report_Cat_Kratikos = allReports[0]; 
String report_Cat_Taktikos = allReports[1];
String report_Cat_Ependyseon = allReports[2];

// Αναφορές Φορέων
String report_Min_Kratikos = allReports[3];
String report_Min_Taktikos = allReports[4];
String report_Min_Ependyseon = allReports[5];
